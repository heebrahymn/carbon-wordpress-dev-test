## Installation

1. Copy the `carbon365-child` folder into:
   `wp-content/themes/`

2. Activate the **Carbon365 Child** theme.

3. Copy the `cc-simple-testimonials` folder into:
   `wp-content/plugins/`

4. Activate the plugin.

5. Create a page titled **Carbon365 Landing Page** with the slug:
   `carbon365`
